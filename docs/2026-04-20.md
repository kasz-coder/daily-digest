## OFFICIAL BLOGS

**Anthropic Engineering**: "Quantifying infrastructure noise in agentic coding evals" — Benchmark scores on agentic coding evals like SWE-bench and Terminal-Bench are less reliable than they appear. Infrastructure configuration alone can produce a 6 percentage point gap — exceeding the margins that separate top models on leaderboards. Key finding: resource headroom matters. Between 1x and uncapped resources, success rates jumped 6 points (p < 0.01). Below 3x, resources fix reliability; above 3x, they actively help agents solve problems. Recommendation: leaderboard differences below 3 points deserve skepticism. [https://www.anthropic.com/engineering/infrastructure-noise](https://www.anthropic.com/engineering/infrastructure-noise)

**Anthropic Engineering**: "Claude Code auto mode: a safer way to skip permissions" — A new mode that uses AI classifiers to approve actions — catching dangerous ones while letting the rest run. Two-layer defense: input-layer probe scans for prompt injection, transcript classifier evaluates each tool call. Results: 0.4% false positive rate on real traffic, 17% miss rate on real overeager actions. Designed to balance autonomy with safety without manual approval fatigue. [https://www.anthropic.com/engineering/claude-code-auto-mode](https://www.anthropic.com/engineering/claude-code-auto-mode)

## PODCASTS

**The MAD Podcast with Matt Turck** — "Anthropic's Felix Rieseberg: Claude Cowork, Mythos, and the SaaS Extinction"

**The Takeaway**: The cost of execution is now near-zero, which fundamentally changes what skills engineers need — shifting from "speaking the computer's language" to "speaking human language."

Felix Rieseberg leads engineering for Claude Cowork at Anthropic, previously at Slack, Stripe, and Notion. He previewed Claude Mythos as a real step-function change. On Cowork: agents can handle complex multi-step tasks for non-technical users across domains like legal, sales, and marketing — and it's already triggering what's being called a "SaaS extinction" in public markets.

On UX for AI agents: trust and taste matter more than interface polish. As execution becomes free, the bottleneck shifts to what you want — directing agents, reviewing output, maintaining systems.

On local computing: the local computer still matters more than Silicon Valley gives it credit for. "Execution is essentially free. If you come to me with 10 different ideas, we can very quickly do all 10."

The skills required will shift from coding to directing, reviewing, and system design.
