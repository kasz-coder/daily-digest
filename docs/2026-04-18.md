## OFFICIAL BLOGS

**Anthropic Engineering** — "Scaling Managed Agents: Decoupling the brain from the hands": The team explains how they solved the classic problem of designing systems for "programs as yet unthought of" by virtualizing agent components (session, harness, sandbox) into abstractions that outlast any particular implementation. Key insight: by decoupling the "brain" (Claude + harness) from the "hands" (sandboxes/tools), each can fail or be replaced independently. This architectural shift reduced p50 time-to-first-token by ~60% and p95 by over 90%. The security boundary ensures tokens are never reachable from the sandbox where generated code runs.
[https://www.anthropic.com/engineering/managed-agents](https://www.anthropic.com/engineering/managed-agents)

**Claude Blog** — "Claude Managed Agents: get to production 10x faster": Launching a suite of composable APIs for building and deploying cloud-hosted agents at scale. Key features: production-grade sandboxing, long-running sessions with persistence, multi-agent coordination, and trusted governance with scoped permissions. Internal testing showed up to 10-point improvement in task success over standard prompting loops. Companies like Notion, Rakuten, Asana, and Sentry are already shipping dramatically faster.
